# 🖼️ Galerie d'Images - Guide d'Utilisation

## 📸 Ajout d'Images Directement

### 🎯 Comment Ça Marche
1. **Cliquez sur le bouton +** en bas à gauche du site
2. **Remplissez** les informations de la voiture
3. **Cliquez sur "Choisir un fichier"** pour l'image
4. **Sélectionnez une image** depuis votre appareil
5. **Aperçu instantané** s'affiche dans le formulaire
6. **Cliquez sur "Ajouter"** pour finaliser

### 📱 Formats Supportés
- ✅ **JPEG** (.jpg, .jpeg)
- ✅ **PNG** (.png)
- ✅ **GIF** (.gif)
- ✅ **WebP** (.webp)
- ✅ **Taille maximale**: 10MB

### 🎨 Fonctionnalités
- 📸 **Upload direct** depuis votre appareil
- 👁️ **Aperçu immédiat** dans le formulaire
- 🔄 **Conversion automatique** en base64
- 💾 **Stockage intégré** dans la base de données
- 🚫 **Pas besoin d'URL externe**
