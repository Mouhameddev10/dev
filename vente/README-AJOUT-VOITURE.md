# 🚗 Ajout Rapide de Voitures - Guide

## 🎯 Comment Ajouter une Voiture

### Étape 1: Accéder à l'Admin
1. **Allez sur** http://localhost:3000
2. **Cliquez sur le bouton +** en bas à gauche (bouton doré avec icône +)
3. **Une fenêtre modale** s'ouvrira avec le formulaire

### Étape 2: Remplir le Formulaire
- **Nom de la voiture**: Ex: "Mercedes G63 AMG 2024"
- **Prix**: En FCFA sans espaces. Ex: "85000000" 
- **Catégorie**: SUV, Sport, GT, ou Électrique
- **Image**: URL d'image (laissez vide pour image par défaut)
- **Description**: Description détaillée de la voiture

### Étape 3: Ajouter au Site
1. **Cliquez sur "Ajouter"**
2. **La voiture apparaît** immédiatement sur le site
3. **Notification de succès** s'affiche

## 💡 Conseils

### Images
- Utilisez des **URL d'images HD**
- Format recommandé: **800x600 pixels**
- Laissez vide pour utiliser l'image par défaut

### Prix
- Entrez le **prix en FCFA** sans espaces
- Ex: 85000000 (pas 85 000 000)
- Le formatage se fait automatiquement

### Description
- Soyez **détaillé** pour attirer les clients
- Mentionnez les **options importantes**
- Utilisez un **langage professionnel**
